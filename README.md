# Пульт руководителя

Личный планировщик для руководителя отдела: в одном месте задачи,
ожидания от команды и проекты. Первая рабочая версия — устанавливаемое
веб-приложение (PWA), которое работает офлайн и ставится на телефон/айпад
как иконка.

## Что уже умеет

- **Захват** (кнопка `+`) — быстро добавить пункт: «моё» или «жду от кого-то».
- **Сводка** — «термометр дня» и секции по тревожности: *Просрочено и горит →
  Пора пнуть → Моё → Ожидания*. Пустой экран — спокойный, без цвета.
- **Закрытие тапом по кольцу** с «периодом благодати» (~6 сек пункт зачёркнут,
  потом исчезает).
- **Undo** — плашка «Отменить» на любое действие (создание, закрытие, удаление).
- **Корзина** — мягкое удаление с восстановлением.
- **Офлайн** — данные хранятся локально (IndexedDB), приложение работает без сети.
- **Темы** — светлая «Бумага» и тёмная «Консоль».

## Как устроено (слоями)

- `src/types.ts` — типы данных: пункт `Item` и запись журнала `Op`.
- `src/store/` — хранилище на IndexedDB (библиотека `idb`).
- `src/core/` — ядро: журнал операций, Undo, период благодати, корзина, правила Сводки.
- `src/components/`, `src/screens/` — экраны и элементы интерфейса.

## Запуск для разработки

```bash
npm install
npm run dev      # локальный адрес появится в консоли
```

## Сборка и просмотр рабочей версии

```bash
npm run build
npm run preview  # покажет адрес вида http://192.168.x.x:4173/
```

## Как открыть на айпаде

1. Айпад и компьютер должны быть в одной Wi-Fi сети.
2. На компьютере: `npm run build`, затем `npm run preview`.
3. В консоли будет строка **Network:** с адресом — например `http://192.168.1.50:4173/`.
4. На айпаде откройте этот адрес в **Safari**.
5. Нажмите **Поделиться** → **На экран «Домой»** — появится иконка «Пульт».
6. Запускайте с иконки: приложение открывается на весь экран и работает офлайн.
